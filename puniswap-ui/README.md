# Incognito Test Dapp

## Getting started

1. Install packages

```
  yarn install
```

2. Start the dev server. And open http://localhost:3000 in your browser.

```
  yarn dev
```

3. Build your product.

```
  yarn prod
```

## Dependencies

- React
- Vite
- Typescript 
- eslint
- Prettier

- Type checking by [vite-plugin-checker](https://www.npmjs.com/package/vite-plugin-checker)
